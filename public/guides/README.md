# Measurement guide images

The daily check-in form (`src/components/patient/DailyCheckInForm.tsx`) shows two
instructional infographics above the reading section. Drop the image files here
with these exact names:

- `bp-technique.png` — "Optimal blood pressure measuring technique" (Hypertension Canada).
- `pulse-check.png` — "Self check: take your own pulse" (find pulse, count 30s, double it).

PNG or JPG both work; if you use JPG, update the `src` extensions in
`DailyCheckInForm.tsx` accordingly. Until the files are added, the figures render
as broken images (the form still works).
